# gin4 EX 위키 레퍼런스 (은하영웅전설 IV EX)

이 문서는 gin7이 기반으로 한 gin4 EX의 상세 게임 시스템을 정리한 것이다.
gin7은 gin4 EX를 온라인 MMO로 확장한 것이므로, 핵심 게임 메카닉스는 동일하다.
Open LOGH 구현 시 이 문서를 참조하되, gin7 매뉴얼(101p PDF)이 우선한다.

## 주요 차이점 (gin4 vs gin7)

- gin4: 싱글/로컬 멀티, 턴제
- gin7: 온라인 MMO, 리얼타임 (24배속)
- gin7에서 추가된 것: 직무권한카드, PCP/MCP, 리얼타임 커맨드, 제안 시스템

이하 위키 내용은 사용자가 제공한 원문 그대로 보존한다.
전면 재작성 시 이 문서의 시스템 설계를 참조할 것.
