## 6. 수정 계획 (우선순위별)

### Phase A — 즉시 수정 (P0)
1. **WebSocket 403 수정**: SecurityConfig에 `/ws/**` permitAll 추가
2. **중원정보 페이지 생성**: v_globalDiplomacy.php 기능 구현 → `/world-diplomacy` 또는 `/global-info`

### Phase B — 주요 수정 (P1)
3. **Troop overlay ID fix**: `map/page.tsx`에서 `foreignTroopCities`/`cityGeneralData` 이름 기반 조회로 변경
4. **맵 년월 표시**: "중원 N年 N月 春/夏/秋/冬" 형식으로 변경
5. **국가 베팅장 페이지**: `/nation-betting` 추가

### Phase C — 커맨드 패러티 검증
6. **FE 커맨드 폼 검증**: 55개 장수 + 38개 국가 커맨드가 command-panel에서 모두 선택 가능한지 확인
7. **커맨드 파라미터 폼**: 각 커맨드의 arg 필드 (도시 선택, 장수 선택, 금액 입력 등) 매핑 확인

### Phase D — 페이지별 상세 검증
8. 각 페이지를 Playwright로 열어 레거시와 1:1 비교
9. 데이터 표시 필드, UI 구성, 기능 동작 확인
