# Open LOGH — gin7 게임 로직 전면 재작성 프롬프트

## 프로젝트 컨텍스트

OpenSamguk(삼국지 웹게임)을 포크하여 은하영웅전설VII(gin7, 2004 BOTHTEC) 기반 웹 MMO로 전환하는 프로젝트.
현재 엔티티 이름과 일부 모델은 LOGH로 변환했으나, **게임 로직 자체는 여전히 삼국지 기반**이다.
이 프롬프트는 삼국지 게임 로직을 gin7 매뉴얼 기준으로 **전면 재작성**하기 위한 것이다.

## 전면 재작성 대상

### 1. 병종 시스템 → 함종 시스템

**삭제해야 할 것:**
- `crewType` (보병/궁병/기병 코드) → 완전 제거
- 병종 상성 시스템 (창>기>궁 등) → 제거
- `UnitModelType = 'infantry' | 'archer' | 'cavalry'` → 제거
- 삼국지 프리셋 캐릭터 (관우, 장비 등) → 제거

**새로 작성:**
- 함정 유닛: 11함종 × I~VIII 서브타입 (ship_stats JSON 데이터 기반)
  - 전함, 고속전함(제국), 순양함, 타격순항함(동맹), 구축함, 전투정모함, 뇌격정모함(제국), 공작함, 수송함, 병원수송함, 양륙함, 민간선
- 기함 유닛: 고유명 기함 + 범용기함 (계급에 따라 결정)
- 육전대 유닛: 장갑병/장갑유탄병/경장육전병 (ground_unit_stats.json 기반)
- 함정 유닛 = 300척 단위, 각 서브타입별 상세 스탯 (장갑, 실드, 무기, 속도, 승무원, 물자적재량)

### 2. 전투 엔진 → gin7 전술전

**삭제:**
- BattleService의 삼국지 수치 비교 자동전투
- 병종 상성 로직
- 삼국지 전투 결과 처리

**새로 작성 (TacticalBattleEngine 확장):**
- 전투 개시: 같은 그리드에 적아 유닛 공존 시 자동 개시
- 에너지 배분: BEAM/GUN/SHIELD/ENGINE/WARP/SENSOR (총합 100, 실시간 변경)
- 무기 시스템: BEAM(중근거리), GUN(중근거리), 미사일(원거리, 물자소비), 전투정(속도저하)
- 진형: 방추/함종별/혼성/삼열 (각각 공격/방어/속도 보정)
- 커맨드레인지서클: 시간경과로 확대, 발령시 0 리셋, 지휘 스탯으로 확대율 결정
- 색적: SENSOR 배분 기반, 거리/유닛종별 정밀도, 전자전 회피
- 요새포: 토르해머/가이에스하켄, 사선 통과 전 유닛 명중, 아군 피격 가능
- 지상전: 육전대 강하, 지상전 박스(30유닛 제한), 행성타입별 참가 가능 유닛
- 점령 후 처리: 직무카드 상실, 수비대 전멸, 긴급출격, 물자접수, 시설접수
- 태세: 항행/정박/주류/전투 (전투모드: 공격↑, 색적↓, 사기↓)
- 전사/부상: 기함 격침 → 부상 → 귀환성 워프

  - 블록 크기 = 함수에 비례
  - 기함 = 별도 3D 모델 (금색/노란색, 함선명 표시)
  - 함종명 영문 표시 (COSMIC-BATTLESHIP, COSMIC-DESTROYER 등)
- **무기 이펙트**: BEAM=노란 레이저 선, 미사일=발사체, 폭발=백색 스프라이트
- **좌측 패널**: 아군 정보 (진영명, 제독명, 함대+부대번호, 함종, 함수 현재/최대, 사기, 무기)
- **우측 패널**: 적군 동일 구조
- **레이아웃**: 상하 2분할 (gin7 도스판 스타일)
  - 상단: 3열 — 좌측 아군 정보 | 중앙 전투 연출 뷰 | 우측 적군 정보
  - 하단: 3D 격자 전술맵 (전체 전장 조감도)

- **상단 좌측 패널** (빨강/갈색 배경):
  - 제독 초상화 (프로필 이미지)
  - 아군 진영명, 제독명
  - 함대번호 + 부대번호
  - 함종 + 함수(현재/최대)
  - 함대사기(현재/최대)
  - 현재 사용 무기명

- **상단 중앙: 전투 연출 뷰**:
  - 검은 우주 배경
  - 교전 중인 양측 함선 — 간단한 3D 모델 또는 2D 스프라이트
  - 무기 발사 이펙트: BEAM=밝은 직선, 미사일=발사체 궤적
  - 피격/폭발 이펙트: 폭발 스프라이트, 파편
  - 기함명 영문 고딕체 표기 (HYPERION, PERUN 등)
  - 교전이 없을 때는 우주 배경만

- **상단 우측 패널** (파랑/남색 배경):
  - 적군 제독 초상화
  - 적군 정보 (좌측과 동일 구조)

- **하단: 3D 격자 전술맵**:
  - 검은 우주 배경 + 파란 격자 바닥
  - 아군(빨강 블록) ← → 적군(파랑 블록) 대치
  - 기함 = 블록 (다른 유닛보다 약간 크게, 기함명 영문 표기)
  - 일반 유닛 블록 크기 = 함수에 비례
  - 기함 위치에 노란 삼각형 마커 (방향 표시)
  - 성운/성역 = 점자무늬 배경 영역

- **공격 연출 (상단 연출 뷰 + 하단 전술맵 동시)**:
  - BEAM = 흰색 직선 레이저 (하단 맵에서도 유닛 간 선으로 표시)
  - 미사일 = 발사체 이동 → 명중 시 폭발
  - 피격 = 유닛 블록 깜빡임 + 함수 감소
  - 상단 연출 뷰에서 확대된 3D/2D 전투 장면 연출

- **React Three Fiber** (기존 Three.js deps 활용)로 구현
  - 상단 연출 뷰: 간단한 함선 스프라이트/로우폴리 + 파티클 이펙트
  - 하단 전술맵: 격자 + 블록 (성능 우선)

## 함대 편성 상세 (gin4 EX 위키 기반)

- 소장: 4개 부대, 중장 이상: 8개 부대 지휘 가능
- 부대 정수: 순양함/구축함/강습양륙함 = 2,000척, 나머지 = 1,000척
- 기함 부대: 반드시 전투함으로 편성, 함대 중앙 삼각형 표시
- 참모: 최대 5명, 사령관보다 낮은 계급, 우주함대사령장관 승인 필요
- 함대 능력치 = 사령관 기본 + 참모 중 최고 수치 보정 (통솔력에 의존)
- 함대 사기: 최대값 = 사령관 통솔 (최소 50), 신규 편성 시 50에서 시작, +5/턴 회복
- 공적 분배: 공적치 높은 쪽이 독식
- 수송함대: 별도 규칙 (왕복 후 자동 해산, 부대 통솔 0, 조병창 성계에서 편성 가능)
- 함대 신설: 수도 성계에서만 가능 (수송함대 제외)
- 편성 변경: 우주함대사령장관 승인, 배치 변경은 사령관 재량

## 군함 상세 (진영별 차이)

### 제국 특성
- 전면 방어력 높음, 속도 느림
- 전함/순양함에 점령 효율(육전대 강하능력) 있음
- 구축함: 레일건 주포 (거리 무관 위력 균일), 유리대포
- 공격항모: 화력 강화했으나 방어/속도 약함

### 동맹 특성
- 속도 빠름, 화력 강함, 방어 약함
- 점령 효율 없음 → 정밀/무차별 폭격으로만 점령 (미사일 소모)
- 고속전함: 최강 함종이나 조병창이 없어 생산 불가 (야리코미 요소)
- 공격항모: 전함급 화력 + 최강 방어력, 돌격 특화

### 무장 시스템
| 무기 | 종류 | 사거리 | 범위 | 위력 | 소속 |
|------|------|--------|------|------|------|
| 광자포 | 빔 | 300 | 80 | 60 | 동맹 전함/고속전함 |
| 중성자빔포 | 빔 | 300 | 80 | 50 | 제국 전함 |
| 중성자포 | 빔 | 240 | 70 | 20 | 공용 |
| 레일캐논 | 레일건 | 300 | 50 | 40 | 제국 구축함 |
| 레이저수폭미사일 | 미사일 | 220 | 160 | 100 | 공용 (제국전함/동맹전함등) |
| 중성자미사일 | 미사일 | 180 | 144 | 80 | 공용 |

- 빔: 최대사거리 70% 근방에서 최대 공격력, 너무 가깝거나 멀면 감소
- 미사일: 거리 무관 위력 균일, 빔보다 강함
- 미사일 부족 시 함대 공격력 크게 감소

## 전술 모드 상세

### 전술 턴 구성 (12턴)
1. 보급 턴 (자동): 물자 갱신
2. 이동 턴: 이동 or 공격 선택, 실시간 동시 이동
3. 수색 턴 (자동): 탐지 범위 내 적 표시
4. 교전 턴: 부대별 공격 대상 지정, 총공격/방어/함재정 발진
5. 점령 턴: 행성 점령 커맨드 (항복권고/정밀폭격/무차별폭격/육전대강하/점거/선동)

### 전술 모드 핵심 규칙
- 플레이어 본인 함대만 직접 지휘 가능
- 작전 총사령관: 최고 계급 → 최고 공적 순으로 자동 결정
- 통신 방해: 정보공작 2000 소비, 스텔스 함대 (점거 필수 콤보)
- 함대 격려: 정치공작 500 소비, 사기 회복
- 위장 함대: 군사공작 1000 소비, 거짓 함대로 유인
- 사기 20 미만: 함대 행동 불가
- 기함 부대 전멸 → 다음 부대가 기함 부대로 대체 (함대는 유지)
- 함대 사령관 사망 → 전략 턴 1턴 후 함대 자동 해산
- 피공격 우선순위: 특정 인물에게 적 전군이 집중

### 성계 지형
- 행성/요새: 점령 대상
- 항성: 근접 시 함선 손실
- 사르갓소 지대: 이동 장애물
- 블랙홀: 끌려가면 전멸 (완전전멸 엔딩)
- 우주조류: 진행방향 왜곡

### 행성 점령 커맨드
| 커맨드 | 지지율 | 경제력 | 방위력 | 최대방위력 | 공적 | 비고 |
|--------|--------|--------|--------|-----------|------|------|
| 항복권고 | - | - | - | - | 높음 | 비용없음, 거의 불발 |
| 정밀폭격 | - | - | ↓ | ↓ | 낮음 | 미사일 소비 |
| 무차별폭격 | ↑ | ↓↓ | ↓↓ | ↓↓ | 최저 | 미사일 대량소비, 융단폭격 |
| 육전대강하 | - | - | ↓ | - | 보통 | 점령효율 필요 (제국 전함/순양함) |
| 점거 | - | - | - | - | 최고 | 군사공작 4000 필요 |
| 선동 | ↓↓ | - | - | - | 보통 | 정보공작 1000, 방공 안 가동 |

### 요새 방위력 기준
- 1,000: 무방비 (1회 정밀폭격으로 점령)
- 5,000~6,000: 주요 행성 (다수 함대 필요)
- 11,000~13,500: 수도급 (아르테미스의 목걸이 포함)
- 30,000: 가르미슈/렌텐베르크 요새
- 50,000+: 이제르론/가이에스부르크 (점거 외 정공법 없음)

### 요새포 위력
- 경X선 빔포: 1,500 (가르미슈/렌텐베르크)
- 아르테미스의 목걸이: 3,000 (하이네센/케니히그라흐)
- 가이에스하켄: 7,000 (가이에스부르크)
- 토르 해머: 10,000 (이제르론) — 1,000척 부대 기준 950척 소멸

## 자금/경제 시스템

- 제국: 군무성 예산 + 통수본부 예산 (분리)
- 동맹: 국방예산 (통합)
- 세금: 90일(전략 30턴)마다, 1/1, 4/1, 7/1, 10/1
- 세율: 기본 30%, 낮으면 지지율↑, 높으면 지지율↓
- 사회간접자본 투자 → 경제력 상승 → 세수 증가
- 행성당 경제력 상한: 50,000
- 차관: 페잔에서 일시불, 이자 10%/분기, 못 갚으면 페잔 엔딩
- 함대 출격 비용: 출격 중 함대 수에 비례하여 자금 소모
- 운영력 높은 장교 = 비용 절감
- 조병창/체재기지/방위기지 유지비 → 후방 불필요 시설 폐쇄 권장
