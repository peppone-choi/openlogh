## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline stat label padding |
| sm | 8px | Compact element spacing, card inner gaps |
| md | 16px | Default card padding, section gaps |
| lg | 24px | Card group spacing, panel padding |
| xl | 32px | Layout column gaps |
| 2xl | 48px | Major section breaks between dashboard zones |
| 3xl | 64px | Page-level top/bottom padding |

Exceptions: none

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (normal) | 1.3 |
| Label | 12px | 400 (normal) | 1.3 |
| Heading | 16px | 700 (bold) | 1.2 |
| Display | 20px | 700 (bold) | 1.2 |

Source: `globals.css` body sets `font-size: 14px; line-height: 1.3`. DungGeunMo pixel font only supports normal (400) and bold (700) weights. 12px used for stat labels, small text, and `<1>` color tag output (0.9em of 14px).

## Color

### System Colors (shadcn CSS variables)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#080c14` (--background) | Page background, surfaces |
| Secondary (30%) | `#0f1520` (--card) / `#151d2c` (--muted) | Cards, popover, sidebar, panels |
| Accent (10%) | `#10b981` (--primary) | Primary CTA buttons, active nav items, selected states |
| Destructive | `#ef4444` (--destructive) | Delete actions, error states, HP damage display |

Accent reserved for: primary action buttons, active sidebar items, selected tab indicators, online status indicators, positive stat changes.

### Phase-Specific Display Labels (Korean, legacy parity)

| Field | Display Label | Format |
|-------|---------------|--------|
| officerCity | 관할 도시 | City name string |
| bill | 봉급 | Integer + "금" suffix |
| specage / specage2 | 특기 개방 | "N세" format |
| killturn | 삭턴 | Integer + "턴" suffix |
| refreshScoreTotal | 벌점 | Integer |
| betray | 배반 | Integer + "회" suffix |
| troopInfo | 부대 상태 | Structured display (name, position) |
| lbonus | 통솔 보너스 | "+N" in cyan color |
| win rate | 승률 | "(killnum / max(warnum, 1)) * 100" + "%" |
| kill ratio | 살상비 | "(killcrew / max(deathcrew, 1)) * 100" + "%" |
| dex level | 숙련도 등급 | Grade name with color (e.g., "달인" in gold) |
| tech level | 기술 등급 | "N등급" format with color |
| injury reduction | 부상 감소 | `Math.floor(stat * (100 - injury) / 100)` applied to displayed stats |
| next execute | 다음 실행 | "N분 남음" format |
| age | 나이 | Integer + "세", colored by retirement proximity |

### Calculated Value Formulas (source of truth)

| Calculation | Formula | Display |
|-------------|---------|---------|
| Injury-reduced stat | `Math.floor(baseStat * (100 - injury) / 100)` | Show reduced value; if injury > 0, color stat orange |
| Win rate | `Math.round((killnum / Math.max(warnum, 1)) * 100)` | "N%" |
| Kill ratio | `Math.round((killcrew / Math.max(deathcrew, 1)) * 100)` | "N%" |
| Dex level | Threshold-based grade from dex value (legacy `formatDexLevel`) | Grade name string with color |
| Tech level | `convTechLevel(tech, startyear, year)` with dynamic max | "N등급" |
| Next execute time | `Math.max(0, Math.floor((turnTime - lastExecuted) / 60000))` | "N분 남음" |
