    - scoreThreat(self, enemy) returns Double: higher = more dangerous. Factors: enemy HP ratio, proximity, ship count, attack stat
    - rankThreats(context) returns List sorted descending by threat score (most dangerous first)
    - shouldRetreat(context) returns true when unit HP ratio < personality retreatHpThreshold OR morale < retreatMoraleThreshold
    - shouldRetreat for AGGRESSIVE personality unit at HP=9% returns true
    - shouldRetreat for AGGRESSIVE personality unit at HP=11% returns false
    - shouldRetreat for CAUTIOUS personality unit at HP=29% returns true
    - shouldRetreat for CAUTIOUS personality unit at HP=31% returns false
    - shouldRetreat for BALANCED personality unit at morale=25 returns true
    - shouldRetreat for BALANCED personality unit at morale=35 returns false
    - scoreThreat scores closer enemy higher than distant enemy (same stats)
    - scoreThreat scores high-HP enemy higher than low-HP enemy (same distance)
  </behavior>
  <action>
    Create `ThreatAssessor.kt` as a pure object (no Spring DI), following UtilityScorer pattern:

    **`data class ThreatScore(val enemyFleetId: Long, val score: Double)`**

    **`fun scoreThreat(self: TacticalUnit, enemy: TacticalUnit): Double`**
    Formula: `threatScore = (enemy.hp / enemy.maxHp) * 40 + (enemy.ships / max(enemy.maxShips, 1)) * 20 + (1.0 - distance / 1000.0).coerceAtLeast(0.0) * 25 + (enemy.attack / 100.0) * 15`
    Where distance = sqrt((self.posX - enemy.posX)^2 + (self.posY - enemy.posY)^2).
    Total scale: 0-100. Higher = more dangerous.

    **`fun rankThreats(ctx: TacticalAIContext): List<ThreatScore>`**
    Score each enemy in ctx.enemies against ctx.unit, sort descending. Filter out retreating enemies.

    **`fun shouldRetreat(ctx: TacticalAIContext): Boolean`**
    Check: `hpRatio < profile.retreatHpThreshold || unit.morale < profile.retreatMoraleThreshold`
    Where hpRatio = unit.hp.toDouble() / unit.maxHp.coerceAtLeast(1).

    **`fun isHighThreat(self: TacticalUnit, enemy: TacticalUnit): Boolean`**
    Returns true if scoreThreat > 60.0. Used by CONQUEST AI to decide "engage weak, bypass strong" per D-01.

    Create `ThreatAssessorTest.kt` with JUnit 5 tests verifying all behavior items above. Use test helper to create TacticalUnit instances with specific HP/morale/position/attack values.
