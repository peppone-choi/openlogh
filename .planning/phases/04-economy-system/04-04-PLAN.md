    ```kotlin
    @Transactional
    fun checkAndTrigger(sessionId: Long) {
        // 중복 방지: 이미 트리거됨
        val world = sessionStateRepository.findById(sessionId).orElse(null) ?: return
        if (world.meta["fezzanEndingTriggered"] == true) return

        val result = fezzanService.checkFezzanEnding(sessionId)
        if (result.triggered && result.dominatedFactionId != null) {
            triggerFezzanEnding(sessionId, result.dominatedFactionId)
            world.meta["fezzanEndingTriggered"] = true
            sessionStateRepository.save(world)
        }
    }
    ```

    processSortieCost(sessionId: Long):
    1. Fleet 조회: fleetRepository.findBySessionId(sessionId)
    2. 출격 중 함대 필터: fleet.meta["isSortie"] == true 또는 fleet.currentUnits > 0인 함대 (Fleet 엔티티 실제 필드 확인 후 적용)
    3. 진영별 그룹화: fleetsByFaction
    4. 각 진영에 대해:
       a. 총 유닛 수 = sum(fleet.currentUnits)
       b. 기본 비용 = totalUnits * BASE_COST_PER_UNIT
       c. 사령관(fleet commander officer, officerRepository에서 fleet.commanderId로 조회) administration으로 할인율 계산:
          - discount = (administration - 50).coerceAtLeast(0) / 100.0 * 0.5  // 최대 50% 절감 (administration=100일 때)
          - finalCost = (baseCost * (1.0 - discount)).toInt()
       d. faction.funds = (faction.funds - finalCost).coerceAtLeast(0)
    5. factionRepository.saveAll(factions)
