    **StrategicPowerScorer.kt** (per D-01):
    ```kotlin
    object StrategicPowerScorer {
        data class StarSystemPower(
            val totalShips: Int,
            val commanderScore: Double,
            val defenseScore: Double,
            val compositeScore: Double,
        )
        fun evaluatePower(fleets: List<Fleet>, officers: List<Officer>, planet: Planet): StarSystemPower
    }
    ```
    - `totalShips` = sum of `fleet.ships` for all fleets at the planet
    - `commanderScore` = average of `(officer.command + officer.leadership) / 2.0` for officers at planet. 0.0 if no officers.
    - `defenseScore` = `(planet.orbitalDefense + planet.fortress).toDouble()`
    - `compositeScore` = `totalShips * 0.5 + commanderScore * 30.0 + defenseScore * 20.0` (raw additive -- normalization happens at comparison time in OperationTargetSelector)

    **FogOfWarEstimator.kt** (per D-02, D-03):
    ```kotlin
    object FogOfWarEstimator {
        const val INTELLIGENCE_THRESHOLD = 70
        const val NOISE_RANGE = 0.4
        fun hasIntelligenceAgent(targetPlanetId: Long, friendlyOfficers: List<Officer>): Boolean
        fun applyFogNoise(truePower: Double, hasAgent: Boolean, rng: Random): Double
    }
    ```
    - `hasIntelligenceAgent`: returns `friendlyOfficers.any { it.planetId == targetPlanetId && it.intelligence >= INTELLIGENCE_THRESHOLD }`
    - `applyFogNoise`: if hasAgent return truePower; else `noise = 1.0 + (rng.nextDouble() * 2 - 1) * NOISE_RANGE`; return `(truePower * noise).coerceAtLeast(0.0)`

    **OperationTargetSelector.kt** (per D-04, D-10):
    ```kotlin
    object OperationTargetSelector {
        data class OperationCandidate(
            val targetPlanetId: Long,
            val objective: MissionObjective,
            val score: Double,
            val estimatedEnemyPower: Double,
        )
        fun selectTargets(
            ownPlanets: List<Planet>,
            enemyPlanets: List<Planet>,
            ownFleetsByPlanet: Map<Long, List<Fleet>>,
            enemyFleetsByPlanet: Map<Long, List<Fleet>>,
            ownOfficersByPlanet: Map<Long, List<Officer>>,
            enemyOfficersByPlanet: Map<Long, List<Officer>>,
            sovereignPersonality: PersonalityTrait,
            friendlyOfficers: List<Officer>,
            rng: Random,
        ): List<OperationCandidate>
    }
    ```
    Logic:
    1. CONQUEST candidates: For each enemy planet with `frontState > 0`, compute enemy power via `StrategicPowerScorer.evaluatePower()`, apply fog noise via `FogOfWarEstimator`. Compute strategic value = `planet.production + planet.commerce + planet.tradeRoute * 100 + planet.population / 100`. Score = `strategicValue / (estimatedEnemyPower + 1.0)`. If AGGRESSIVE sovereign, multiply CONQUEST scores by 1.5.
    2. DEFENSE candidates: For each own planet with `frontState > 0`, compute own power. Find maximum adjacent enemy power (from enemy planets). If enemyPower > ownPower * 0.7, it's threatened. Score = `enemyThreat / (ownPower + 1.0)`. If DEFENSIVE sovereign, multiply DEFENSE scores by 1.5.
    3. SWEEP candidates: For each own planet where `enemyFleetsByPlanet[planet.id]` is non-empty, score = enemy fleet ships count. If CAUTIOUS sovereign, multiply SWEEP scores by 0.8 (less aggressive sweeping).
    4. Return all candidates sorted by score descending.

    - Sort fleets by `fleetPower(fleet)` descending
    - Greedily add fleets until cumulative power >= target or fleets exhausted
    - Return selected IDs and total allocated power
    - If no fleets, return empty AllocationResult
