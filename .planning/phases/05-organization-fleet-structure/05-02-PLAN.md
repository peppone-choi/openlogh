    Create FormationCapService as @Service with constructor-injected PlanetRepository and FleetRepository.

    Key methods:
    ```kotlin
    /** Calculate max formable units per type for a faction based on total population */
    fun getFormationCaps(sessionId: Long, factionId: Long): Map<UnitType, FormationCap>

    /** Check if a faction can form one more unit of this type */
    fun canFormUnit(sessionId: Long, factionId: Long, unitType: UnitType): Boolean

    /** Get total population across all faction planets */
    fun getFactionPopulation(sessionId: Long, factionId: Long): Long
    ```

    Data class:
    ```kotlin
    data class FormationCap(val current: Int, val max: Int) {
        val available: Int get() = (max - current).coerceAtLeast(0)
    }
    ```

    Population calculation: Sum all `planet.population` for the faction. The population field stores values where population=100000 means 1 billion (100,000 x 10,000 = 1,000,000,000). So threshold for 1 fleet = population sum >= 100,000.

    For fleet/transport: max = totalPop / 100_000 (integer division)
    For patrol/ground: max = (totalPop / 100_000) * 6

    Use PlanetRepository.findByFactionId to get planets, FleetRepository.findByFactionId to count existing units grouped by unitType.

    Add to FleetRepository: `fun countByFactionIdAndUnitType(factionId: Long, unitType: String): Long`
