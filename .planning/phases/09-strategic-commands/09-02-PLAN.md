# Phase 9 Plan 02: Logistics & Supply Chain Commands

## Objective
Add gin7 logistics commands: supply allocation (planet warehouse to fleet), fleet resupply, production oversight, and reorganization.

## Tasks

### Task 1: Implement logistics and supply commands
type="auto"

**New officer commands:**
1. `물자배분` (Supply Allocation) - LOGISTICS group, PCP pool
   - Transfer supplies from planet warehouse to officer's personal stores
   - Requires officer on owned planet with supplies
   - Amount specified by arg, capped by planet availability
   - Duration 300s, costs 1 PCP

2. `함선보급` (Fleet Resupply) - LOGISTICS group, MCP pool
   - Transfer ships from planet garrison to officer's fleet
   - Planet must have production facilities and available ships
   - Costs 1 MCP, duration 300s

3. `함대재편` (Fleet Reorganization) - LOGISTICS group, MCP pool
   - Change ship class of officer's fleet
   - Requires being on owned planet with production facilities
   - Training drops by 20% on class change, morale unchanged
   - Costs 1 MCP, duration 600s

4. `생산감독` (Production Oversight) - LOGISTICS group, PCP pool
   - Boost planet's production output for current period
   - Uses administration stat to increase production tick
   - Costs 1 PCP, duration 300s

**New faction command:**
5. `예산편성` (Budget Allocation) - POLITICS group, PCP pool
   - Set faction tax rate and conscription rate
   - Only faction leaders, costs 2 PCP
   - Duration 600s

**Done criteria:**
- All 5 commands compile and are registered
- Commands mapped in PositionCardRegistry
