# Phase 9 Plan 03: Intelligence & Reconnaissance Commands

## Objective
Add gin7 intelligence commands: reconnaissance, communication jamming, and enhanced espionage.

## Tasks

### Task 1: Implement intelligence and recon commands
type="auto"

**New officer commands:**
1. `정찰` (Reconnaissance) - INTELLIGENCE group, MCP pool
   - Reveal enemy fleet positions and planet info in target system
   - Uses intelligence stat for success probability
   - Costs 1 MCP, duration 300s

2. `통신방해` (Communication Jamming) - INTELLIGENCE group, MCP pool
   - Disrupts enemy communications in target system
   - Reduces enemy command point recovery in affected area
   - Costs 2 MCP, duration 600s

3. `점거` (Occupation/Seizure) - OPERATIONS group, MCP pool
   - Occupy an undefended or weakly defended planet
   - Requires fleet presence, no strong garrison
   - Changes planet ownership on success
   - Costs 2 MCP, duration 900s

**Done criteria:**
- All 3 commands compile and are registered
- Mapped correctly in PositionCardRegistry
