<!-- BaseCommand 계약 -->
```kotlin
abstract class OfficerCommand(general: Officer, env: CommandEnv, arg: Map<String, Any>?) : BaseCommand(general, env, arg)

// 필수 override:
abstract val actionName: String
abstract fun getCost(): CommandCost           // funds/supplies 비용
open fun getCommandPoolType(): StatCategory   // PCP(기본) or MCP
abstract fun getPreReqTurn(): Int             // waitTime (틱 단위) → 실행 전 대기
abstract fun getPostReqTurn(): Int            // duration (틱 단위) → 실행 후 쿨다운
abstract suspend fun run(rng: Random): CommandResult
```

<!-- commands.json 핵심 값 (작전/병참) -->
작전커맨드 MCP:
- 워프항행: cpCost=40, waitTime=0(variable), duration=0(variable)
- 연료보급: cpCost=160, waitTime=8, duration=48~960
- 성계내항행: cpCost=160, waitTime=8, duration=0
- 군기유지/항공훈련/육전훈련/공전훈련/육전전술훈련/공전전술훈련: cpCost=80, waitTime=0, duration=0
- 경계출동: cpCost=160, waitTime=24, duration=0
- 무력진압: cpCost=160, waitTime=24, duration=0
- 분열행진: cpCost=160, waitTime=24, duration=0
- 징발: cpCost=160, waitTime=24, duration=0
- 특별경비: cpCost=160, waitTime=0, duration=24
- 육전대출격: cpCost=80, waitTime=0, duration=0
- 육전대철수: cpCost=80, waitTime=0, duration=0

병참커맨드 MCP:
- 완전수리/완전보급/재편성/보충/반출입/할당: cpCost=160, waitTime=0, duration=0
