    Export `function canCommandUnit(myOfficerId: number, myHierarchy: CommandHierarchyDto | null | undefined, targetUnit: TacticalUnit): GatingResult`.

    Logic per D-12 (from 14-RESEARCH Section 12):
    - If `!myHierarchy` → `{ allowed: false, reason: 'OUT_OF_CHAIN' }`

    - If `mySubFleet && mySubFleet.memberFleetIds.includes(targetUnit.fleetId)` → `{ allowed: true, reason: null }`
    - Otherwise → `{ allowed: false, reason: 'OUT_OF_CHAIN', message: '지휘권 없음 — Shift+클릭으로 상위자에게 제안하기' }`
