    Internal `Bucket({ id, title, children, empty })` uses `useDroppable({ id })`, applies `bg-accent` on `isOver`, renders empty hint "유닛을 이곳으로 끌어다 놓아 배정합니다." when `empty`.

    Main component `useMemo` selects mySide from `myUnit.side`, picks `attackerHierarchy`/`defenderHierarchy`, then builds `myBuckets`:
    - Iterate `hierarchy.subFleets` up to 8 slots → bucket id `sub-{commanderOfficerId}`, title `"{commanderName} ({rank}) — {n}/60"`
    - Pad remaining slots with labels `['부사령관', '참모장', '참모 1', '참모 2', '참모 3', '참모 4', '참모 5', '참모 6']` as empty id=`slot-{i}`

    - Append `{ id: 'unassigned', title: '미배정 유닛 — {n}', ... }`

    Sensors: `useSensors(useSensor(PointerSensor, { activationConstraint: { distance: 4 } }), useSensor(KeyboardSensor))`.

    Export a pure helper `createDragEndHandler(sessionId: number, myOfficerId: number)` that returns `(event: DragEndEvent) => void`. Inside:
    - If `!over || !active.data.current` return
    - `targetId = String(over.id)`
    - `commandCode = targetId === 'unassigned' ? 'ReassignUnit' : 'AssignSubFleet'`
    - `subFleetCommanderId = targetId.startsWith('sub-') ? Number(targetId.slice(4)) : null`
    - `publishWebSocket('/app/command/' + sessionId + '/execute', { officerId: myOfficerId, commandCode, args: { targetFleetId: active.data.current.fleetId, subFleetCommanderId } })`
    - `toast.success(...)` with Korean acknowledgement

    Return `null` if `!battle || !mySide`. Render `<ResponsiveSheet open={open} onOpenChange={onOpenChange} side="right" title="분함대 편성">` containing subtitle row, then `<DndContext sensors={sensors} collisionDetection={closestCenter} onDragEnd={createDragEndHandler(sessionId, myOfficerId)}>` wrapping the bucket list + `<DragOverlay />`.

    Per-chip rendering: compute `commanderUnit` by looking up `hierarchy.fleetCommander` in mySideUnits; call `canReassignUnit(u, phase, hierarchy, commanderUnit)`; pass `disabled={!gate.allowed}` and `disabledReason={gate.message}` to `SubFleetUnitChip`.
