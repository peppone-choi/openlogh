# Plan 23-05: Gin7EconomyService.updateFactionRank

## Objective

Port legacy `updateNationLevelEvent` → `updateFactionRank`. Recalculates each faction's rank (방랑군 / 도위 / ... / 황제) based on military_power, planet count, and population thresholds. Triggered annually.

## Legacy reference

The legacy EconomyService has `NATION_LEVEL_NAME` constants and `getNationLevelName(level)`. The rank calculation logic is in the upstream (pre-stub) `updateNationLevelEvent` body — read via:

```bash
git show a7a19cc3~1 -- backend/game-app/src/main/kotlin/com/opensam/engine/EconomyService.kt | awk '/updateNationLevel/,/^\s*\}/'
```

Legacy ranks scale with `military_power` thresholds (방랑군 → 도위 at 1000 MP, etc. — exact thresholds in upstream).
