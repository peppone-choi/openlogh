# Plan 23-06: Gin7EconomyService.updatePlanetSupplyState (move from legacy)

## Objective

Move the existing `EconomyService.updateCitySupplyState` + `updateCitySupply` logic into `Gin7EconomyService.updatePlanetSupplyState`. Unlike 23-01..23-05, this method ALREADY HAS functional LOGH code — we just relocate it and update call sites.

Legacy source: `EconomyService.kt:134-353`.

## Key mechanics (already in LOGH)

1. Build BFS from faction capital through connected supplied planets (map-connectivity)
2. Mark supplied planets `supplyState = 1`, isolated `supplyState = 0`
3. Isolated planets suffer 10% decay on population/approval/production/commerce/security/orbital_defense/fortress
4. Isolated officers lose 5% ships/morale/training
5. Planets below approval 30 defect to neutral (factionId = 0)
