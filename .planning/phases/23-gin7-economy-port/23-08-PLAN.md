# Plan 23-08: Gin7EconomyService.processDisasterOrBoom

## Objective

Port the legacy disaster/boom event generator. Each month has a low probability of generating a random disaster (plague, earthquake, famine) or boom (harvest, trade surge) at a random planet, applying resource modifiers + emitting a GameEvent log entry.

This is the largest single port (~164 lines). The disaster/boom TABLE (state codes + Korean titles + body templates) must be ported verbatim from the legacy `DisasterOrBoomEntry` table.

## Legacy reference

`EconomyService.kt:439-602` contains:
- `DisasterOrBoomEntry` data class (line 69-73)
- `processDisasterOrBoom(world)` method
- Probability table and state transitions
- Korean event log templates
