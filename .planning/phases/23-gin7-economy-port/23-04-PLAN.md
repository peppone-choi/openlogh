## Objective

Implement the faction → officer salary payment step. Each faction pays its officers monthly salary computed via the Phase 22-01 `getBillFromDedication(dedication)` formula. The method must be callable from the scheduled income event (month 1) so that it runs ONCE per year.

## Dependencies

- Wave 1 (23-01, 23-02, 23-03) must land first — this plan reads the per-resource income API
- Phase 22-01 `FactionAI.getBillFromDedication` helper MUST be made public/visible to Gin7EconomyService (either by exposing it, moving it to a shared location, or duplicating it in Gin7)

3. Faction with insufficient funds: salary still paid; funds can go negative (matches legacy)
