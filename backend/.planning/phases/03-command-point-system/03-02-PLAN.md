  <name>Task 1: Create CpService with deduction, cross-use, and recovery logic</name>
  <files>

  <behavior>
    - deductCp with sufficient primary pool: deducts exactly cost from primary pool, returns success
    - deductCp with insufficient primary but sufficient cross pool: deducts cost*2 from cross pool, returns success with cross-use flag
    - deductCp with insufficient both pools: returns failure with error message
    - deductCp cross-use: MCP command cost 3, officer has mcp=1 pcp=10 -> deducts 6 from pcp, returns success
    - regeneratePcpMcp: pcp recovers by floor((politics+administration)/20)+1, capped at pcpMax
    - regeneratePcpMcp: mcp recovers by floor((command+mobility)/20)+1, capped at mcpMax
    - regeneratePcpMcp: already-at-max pool stays at max
    - getExpStatsForPool PCP: returns [LEADERSHIP, POLITICS, ADMINISTRATION, INTELLIGENCE]
    - getExpStatsForPool MCP: returns [COMMAND, MOBILITY, ATTACK, DEFENSE]
  </behavior>
  <action>
    Create `CpService.kt` as a Spring `@Service` in `com.openlogh.service`:

    ```kotlin
    data class CpDeductionResult(
        val success: Boolean,
        val crossUsed: Boolean = false,
        val actualCost: Int = 0,
        val poolUsed: StatCategory? = null,
        val errorMessage: String? = null,
    )
    ```

    Methods:
    1. `deductCp(officer: Officer, cost: Int, poolType: StatCategory): CpDeductionResult`
       - If poolType is PCP and officer.pcp >= cost: deduct from pcp, return success
       - If poolType is MCP and officer.mcp >= cost: deduct from mcp, return success
       - Cross-use: if primary insufficient, check other pool for cost*2. If sufficient, deduct cost*2 from other pool, return success with crossUsed=true
       - If neither pool sufficient: return failure with message "커맨드 포인트가 부족합니다. (필요: {cost} {poolName}, 보유: PCP {pcp}/{pcpMax}, MCP {mcp}/{mcpMax})"

    2. `regeneratePcpMcp(officer: Officer)` — no return, mutates officer fields
       - pcpRegen = floor((officer.politics + officer.administration) / 20.0).toInt() + 1
       - mcpRegen = floor((officer.command + officer.mobility) / 20.0).toInt() + 1
       - officer.pcp = (officer.pcp + pcpRegen).coerceAtMost(officer.pcpMax)
       - officer.mcp = (officer.mcp + mcpRegen).coerceAtMost(officer.mcpMax)

    3. `getExpStatsForPool(poolType: StatCategory): List<OfficerStat>` — returns the stats that gain exp when that pool type's command is used (delegates to OfficerStat.pcpStats/mcpStats)

    Write tests in CpServiceTest using JUnit 5. CpService is stateless (no injected dependencies), so tests instantiate it directly.
